## Amplify Auth

- https://docs.amplify.aws/sdk/auth/getting-started/q/platform/ios/
- https://docs.amplify.aws/cli/auth/import/

1. amplify init
   amplify フォルダと各種設定ファイル、実際の amplify プロジェクトが生成される
2. amplify import auth
   既存の cognito をインポートする
3. amplify push
